# mk switch
built in pure css

web component built in lit-element

to include as a web component add the script `<script src="https://switch.fallproject.org/switch.lit.js">`
and use the element `<mk-switch></mk-switch>`

## Properties

| **Name** | **Type** | **Example** |
| -------- | -------- | :---------: |
| Name     | String   |  `switch1`  |
| Checked  | Boolean  |   `true`    |
